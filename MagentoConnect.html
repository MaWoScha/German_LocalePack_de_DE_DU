Hinweis / Hint / Aviso

Magento kommt in der Grundausstattung mit amerikanischem Englisch daher. Wer britisches Englisch nutzen möchte, sollte das Sprachpaket LocalePack_en_GB installieren.

Magento comes in its basic configuration with American English. Anyone wishing to use British English, should install the language pack LocalePack_en_GB.

Magento viene en su configuración básica con Inglés americano. Cualquier persona que desee utilizar Inglés británico, debe instalar el paquete de idioma LocalePack_en_GB.
